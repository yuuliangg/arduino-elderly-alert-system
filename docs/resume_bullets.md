## Full Project Entry
Arduino Smart Elderly Alert System  
Arduino C++ • Ultrasonic Sensor • Servo Motor • Buzzer • LED

- Built a sensor-based elderly safety prototype using an ultrasonic sensor, push button, LED, buzzer, and servo motor to support manual and automatic emergency alerts.
- Programmed multi-stage alert logic in Arduino C++, escalating from LED warning to servo nudge and full emergency mode when no movement is detected.
- Implemented distance thresholding, manual emergency activation, reset logic, and Serial Monitor debugging to improve reliability and reduce false alerts.
